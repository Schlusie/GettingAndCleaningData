GettingAndCleaningData

This repository is the course project of the Getting and Cleaning Data Coursera course.

You have to run the run_analysis.R-file. This subsequently loads the used data sets which are in getdata_projectfiles_UCI HAR Dataset.zip without extracting.

After running the code two tidy data sets (as described in the instructions) are saved:

  1. tidy_dataset1.txt: consists of a 10299x81 with ;-seperation, header and no row.names
  2. tidy_dataset2.txt: consists of a 180x81 with ;-seperation, header and no row.names

You can load the data via the codes:

tidy1 <- read.table("tidy_dataset1.txt", sep=";", header=TRUE)
tidy2 <- read.table("tidy_dataset2.txt", sep=";", header=TRUE)

have fun!
